# base-meme-coin-launchpad

Fair-launch meme coin platform on Base with bonding curves and anti-snipe protections.

## Overview
Educational starter for building fair and transparent meme coin launches on Base.

## Tech Stack
- Solidity
- Hardhat
- Uniswap V3 / V2 for liquidity

## Getting Started
```bash
git clone https://github.com/Proxtox/base-meme-coin-launchpad.git
cd base-meme-coin-launchpad
npm install
```

## Key Files
- `contracts/MemeCoinLauncher.sol` - Bonding curve style contribution + launch logic
